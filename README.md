# NetGen Pro - DPDK Edition v2.0

High-Performance Network Traffic Generator with 10+ Gbps capability

## Quick Start

```bash
sudo bash scripts/install.sh
cd web && source venv/bin/activate
sudo python3 dpdk_control_server.py
```

Access: http://localhost:8080

## Features
- 10+ Gbps throughput
- UDP/TCP/ICMP protocols  
- Multi-profile support
- Web UI with real-time stats
- Profile management
- Test history
- REST API + WebSocket

See QUICK-START.md for complete guide
